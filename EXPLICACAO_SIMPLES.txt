═══════════════════════════════════════════════════════════
  EXPLICAÇÃO SIMPLES - PASSO A PASSO
═══════════════════════════════════════════════════════════

VOCÊ ESTÁ AQUI:
───────────────
✅ ZIP foi criado
✅ ZIP foi enviado para o GitHub
✅ ZIP foi extraído na sua pasta

O PROBLEMA:
────────────
O Render (servidor) não consegue usar ZIP.
Ele precisa dos ARQUIVOS DESCOMPACTADOS.

O QUE FAZER:
────────────
1. Deletar o ZIP do GitHub
2. Enviar os arquivos descompactados para o GitHub
3. Fazer deploy no Render

═══════════════════════════════════════════════════════════
PASSO 1 - DELETAR O ZIP DO GITHUB
═══════════════════════════════════════════════════════════

1. Abra o navegador
2. Vá em: https://github.com/emanuelcastro592-svg/emanuel
3. Você vai ver o arquivo: NEW-APP-LIGHT.zip
4. CLIQUE NELE (no nome do arquivo)
5. Vai abrir uma página mostrando o arquivo
6. Procure um ícone de LIXEIRA (Delete)
7. CLIQUE na lixeira
8. Vai pedir para confirmar - digite: NEW-APP-LIGHT.zip
9. CLIQUE em "Commit changes"
10. PRONTO! O ZIP foi deletado.

═══════════════════════════════════════════════════════════
PASSO 2 - ENVIAR OS ARQUIVOS DESCOMPACTADOS
═══════════════════════════════════════════════════════════

1. Ainda no GitHub, na mesma página do repositório
2. Procure um botão VERDE que diz "Add file"
3. CLIQUE nele
4. Vai abrir um menu - escolha "Upload files"
5. Agora você precisa abrir a pasta do projeto:
   - Abra o Explorador de Arquivos do Windows
   - Vá em: C:\Users\User\Downloads\NEW APP
6. Selecione os arquivos e pastas:
   - Clique na pasta "server" (segure CTRL e clique)
   - Clique na pasta "client" (segure CTRL e clique)
   - Clique em "package.json" (segure CTRL e clique)
   - Clique em "package-lock.json" (segure CTRL e clique)
   - Clique em "README.md" (segure CTRL e clique)
   - Clique em "Procfile" (segure CTRL e clique)
   - Clique em "render.yaml" (segure CTRL e clique)
   - Clique em ".gitignore" (segure CTRL e clique)
   - Clique em "ecosystem.config.js" (segure CTRL e clique)
7. ARRASTE esses arquivos para a área de upload do GitHub
   (a área onde diz "Drag files here to add them to your repository")
8. Role a página do GitHub para BAIXO
9. Vai ter um campo escrito "Commit changes"
10. Digite: Initial commit - all files
11. CLIQUE no botão verde "Commit changes"
12. AGUARDE terminar (pode demorar alguns minutos)

═══════════════════════════════════════════════════════════
PASSO 3 - FAZER DEPLOY NO RENDER
═══════════════════════════════════════════════════════════

1. Abra o navegador em: https://dashboard.render.com
2. Faça login
3. Vai aparecer seu serviço (personal-trainer-api)
4. CLIQUE nele
5. Procure um botão "Manual Deploy"
6. CLIQUE nele
7. Vai aparecer "Deploy latest commit"
8. CLIQUE nele
9. AGUARDE o deploy terminar (pode demorar 5-10 minutos)

═══════════════════════════════════════════════════════════
RESUMO ULTRA SIMPLES:
═══════════════════════════════════════════════════════════

1. GitHub → Delete o ZIP
2. GitHub → Upload dos arquivos descompactados
3. Render → Deploy

PRÓXIMOS PASSOS:
────────────────

1. REGISTRAR O DOMÍNIO (se ainda não tem)
   → Registro.br, GoDaddy, Namecheap, etc.
   → Custo: ~R$ 30-50/ano

2. TER UM SERVIDOR NA NUVEM
   → DigitalOcean, AWS, Azure, etc.
   → Ou servidor local com IP público fixo
   → Custo: ~R$ 20-100/mês (se na nuvem)

3. CONFIGURAR DNS
   → No painel do domínio, apontar para IP do servidor
   → Registro A: personalsuper2.com.br → IP_DO_SERVIDOR
   → Registro A: www.personalsuper2.com.br → IP_DO_SERVIDOR

4. CONFIGURAR SSL/HTTPS
   → Let's Encrypt (grátis) ou certificado pago
   → Usar Nginx como proxy reverso (recomendado)

5. FAZER BUILD E DEPLOY
   → cd client && npm run build
   → Enviar código para servidor
   → Configurar PM2 no servidor

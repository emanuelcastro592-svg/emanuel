Na página do GitHub, você verá uma caixa AZUL CLARA que diz:

"Quick setup — if you've done this kind of thing before"

DENTRO DESSA CAIXA AZUL, procure por:

"Get started by creating a new file or uploading an existing file."

TEM UM LINK AZUL que diz:
→ "uploading an existing file"
→ (está em negrito/azul, é clicável)

CLIQUE NESSE LINK! ✅

═══════════════════════════════════════════════════════════

"Add file" ou "Create new file"

Clique nele, depois procure por:
"upload files" ou "uploading an existing file"
